# Vue ToDo List

## About
Project of a CRUD ToDo list made in vue.js

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
## Docker
1. docker build -t todolist:v1 .
2. docker run --name todolist -dp 80:80 todolist:v1
3. Access <host_ip>
